# A Brighter Summer Day

阳光灿烂的日子（雾）  
不过本片确实有与其相似的地方  
同样的少年遇到了一个灿烂的女孩  
少年的爱都笼罩在那个更强的他者（牯岭街之哈尼之风光&romance、校医之于小明母亲的切身好处、小马之家境乃至滑头之武力；于阳光灿烂而言，主角对比刘忆苦似乎一无是处，于是只好陷入幻想的英勇中，最终在现实的一地鸡毛中行懦弱苟且之事却被米兰反杀）阴影之下  
名曰阳光灿烂，实则为之炫目 深感卑微阴湿
